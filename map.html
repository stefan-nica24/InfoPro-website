Map-urile din STL C++

Cuprins

  • Declarare
  • Adăugarea unui element în map
  • Parcurgere
  • Tabel cu toate funcțiile din cadrul map-urilor


Map-urile sunt containerele asociative în STL care sunt folosite pentru a stoca elementele sub formă de pereche <Key,Value>. Pe plan intern, elementele dintr-un map sunt întotdeauna sortate după key-ul acesteia. Maps sunt implementate în principal ca arbori binari de căutare. Funcția at() este folosită pentru a returna referința la elementul asociat cu key-ul k. Map este inclus in biblioteca cu același nume.


In limbaj comun, se poate folosi ca un vector de frecventa pentru a numara o anumita cheie pe valoarea corespunzatoare.

1. Cum se declară map-urile in C++ 


std::map<int, int> nume_map;

Structura generală a unui astfel de container este pre map <tip_date, tip_date> nume.

2. Adăugarea unui element


Pentru a adăuga elemente în acest tip de date se va folosi operatorul []. 

ex: m[5]=1;

3. Parcurgerea map-ului


Pentru a parcurge map-ul nefiind un container secvențial, nu se poate accesa cu random-access iterator și va fi nevoie de un iterator specific map-ului care să îl parcurga pe segmentul begin->end. 

std::map<int,int> m; 
m[1]=2; 
m[2]=5; 
m[1]++; 
m[3]=10; 
std::map<int,int>::iterator it=m.begin(); 
for(; it!=m.end(); it++) 
    std::cout<<it->first<<' '<<it->second<<std::endl;
//sau
for(auto it : m) 
    std::cout<<it.first<<' '<<it.second<<std::endl;

Va afișa: 

1 3 
2 5 
3 10

4. Toate funcțiile importante ale clasei std::map din C++

Funcția

 Descriere
insert() Adaugă un element cu o anumită specificație in map
count() Returnează numărul de elemente cu o anumită valoare în map
size() Returnează numarul de elemente din map
equal_range() Returnează un iterator de perechi care include toate elementele din containter ce au "cheia" echivalenta cu k
rend() Returnează un iterator-inversat care face trimitere la capătul inversat al map-ului
rbegin() Returnează un iterator-inversat care face trimitere la ultimul element din map
operator[] Accesează elementul de pe o poziție specifică
at() Returnează o referință la un element asociat indexului k
find() Returnează un iterator la elementul cu valoarea-cheie 'k' în map, dacă este găsit. Altfel returnează .end()
crbegin() Returnează un iterator-inversat constant care se referă la ultimul element din container
crend() Returnează un iterator-inversat constant care indică elementul teoretic înainte de primul element din container
cbegin() Returnează un iterator constant care se referă la primul element din container
cend() Returnează un iterator constant care indică elementul teoretic care urmează ultimul element din map
erase(iterator) Șterge elementul de pe o poziție anume
swap() Schimbă elementele unui map cu cele ale altui map
clear() Șterge toate elementele din vector
emplace() Construiește și inserează un element intr-un map
begin() și end() begin() returnează un iterator la primul element din map. end() returneazp iteratorul teoretic ce urmează ultimul element din map
empty() Verifică dacă map-ul este gol
