Vectorii STL din C++

Cuprins

  • Declarare
  • Adăugarea unui element în vector
  • Parcurgerea vectorului
  • Iteratori în cadrul vectorilor
  • Constructor și alocare
  • Ștergeri și inserări
  • Redimensionare
  • Tabel cu toate funcțiile din cadrul vectorilor


Vectorii sunt containăre alocate dinamic, având capacitatea de a se redimensiona automat atunci când un element este inserat sau șters. Stocarea lor este gestionată automat de container. Elementele vectoriale sunt plasate în stocare contiguă, astfel încât să poată fi accesate și traversate folosind iteratoare. Eliminarea ultimului element durează doar timp constant, deoarece nu are loc nicio redimensionare. Inserarea și ștergerea la început sau la mijloc este liniară în timp.

Pentru declararea unui vector trebuie inclusă biblioteca <vector> unde se află funcții de manipulare a acestui container.

Inițial vectorul este gol având 0 elemente. Se obișnuiește ca în lucrul cu tablouri statice din C să se prealoce numărul de poziții pentru cât se crede că va fi nevoie și folosirea poziților de la 1.

1. Cum se declară vectorii în C++ 


std::vector<tip_dată> nume_vector;

Pentru declararea unui vector trebuie inclusă biblioteca <vector> unde se află funcții de manipulare a acestui container.

2. Adăugarea unui element 


std::vector<int> v;
for(int i=1; i <= 5; ++i)
     v.push_back(i);
cout<<v.size();      //5

Adăugarea unui nou element în vector se face la sfârșit, cu funcția push_back().

3. Parcurgerea vectorului și afișarea lui 


for(int i=0; i < v.size(); ++i) 
    cout<<v[i]<<' '; | cout<<v.at(i)<<' ';

Funcția .at() este un membru de random-access specific vectorului care înlocuiește operatorul [] având avantajul că dacă se întreabă de o poziție invalidă aruncă excepție.

4. Parcurgerea vectorului cu ajutorul unui iterator


Ce este un Iterator?

Un iterator este un pointer la adresă, specific fiecărui tip de container pentru parcurgerea lui din adresă în adresă.


Cum funcționeaza?

Iteratorul se declară pentru fiecare tip de container și fiecare tip de date a containerului. NU se pot folosi pentru alte containere de alt tip. După declarare se atribuie adresa de început .begin() și se se incrementează, vectorul fiind declarat secvential se va opri atunci cand se termină segmentul pe care este declarat vectorul.


Fiecare vector are 2 membrii begin(), end()

Funcția begin() are adresa de început a vectorului și end() adresa de sfârșit a vectorului care se află cu o poziție după ultimul element și ea este o poziție invalidă care nu face parte din vector. Fiecare element se parcurge prin dereferinta iteratorului.

Atenție! Orice operațiune de modificare a secvenței din interiorul vectorului va invalida iteratorul deoarece spațiul va fi realocat.


Exemplu 

vector<int> v;
vector<int>::iterator it;
for(it=v.begin(); it!=v.end(); ++it) 
    cout<<*it<<' ';


Mai Mult!

Se poate forța prin instrucțiunea auto să determine ce fel iterator este nevoie pentru a parcurge containerul. 

vector<int> v;
for(auto i : v)
    cout<<i<<' ';

5. Constructor și alocare

Vectorul nu este obligatoriu să fie gol el poate să fie declarat cu un anumit număr de poziții exact cum s-ar face la cei statici și să fie modificate pe parcurs. De asemenea el are în componență un constructor care spune cu ce să umple fiecare poziție. 

vector<int> v(100);    //Vector cu 100 de elemente cu pozitiile 0-99 si valoarea 0 implicită.
vector<int> v(100, 1);    //Vector cu 100 de elemente cu pozitiile 0-99 si valoarea 1.

6. Ștergeri și inserări

Acest lucru se poate realiza în aproximativ o(n), limitata de modalitate de structurare a vectorului folosind .erase(), .insert().

Exemplu: 

v.insert(v.begin()+2,10); 
v.erase(v.begin()+2);

7. Redimensionarea

Funcția resize() permite redimensionarea unui vector și are următoarele forme:

  • v.resize(n); (1)
  • v.resize(n , x); (2)

Vectorul V se redimensionează încât să aibă n elemente:

  • Dacă n < V.size(), în vector vor rămâne primele n elemente, celelalte vor fi șterse.
  • Dacă n > V.size(), în vector se vor adăuga n - V.size() elemente egale cu x , în cazul (2) sau egale cu valoarea implicită pentru tipul elementelor din vector, în cazul (1). Această valoare este 0 pentru tipurile numerice.

8. Toate funcțiile importante ale clasei std::vector din C++

Funcția

 Descriere
push_back() Adaugă un element la finalul vectorului
pop_back() Scoate ultimul element al vectorului
size() Returnează dimensiunea curentă a vectorului
max_size() Returnează numărul maxim de elemente pe care le poate ține vectorul
resize() Schimbă capacitatea vectorului
empty() Verifică dacă vectorul este gol
operator[] Accesează elementul de pe o poziție specifică
at() Accesează elementul dintr-o anumită poziție, cu verificarea limitelor
front() Accesează primul element din vector
back() Accesează ultimul element din vector
begin() Returnează un iterator care face trimitere la primul element din vector
end() Returnează un iterator care face trimitere la ultimul element din vector
insert() Inserează un element la o poziție anume
erase(iterator_first, iterator_last) Șterge elementele dintr-un interval anume din vector
erase(iterator) Șterge elementul de pe o poziție anume
swap() Schimbă elementele unui vector cu cele ale altui vector
clear() Șterge toate elementele din vector
emplace() Construiește și inserează un element intr-un vector
emplace_back() Construiește și inserează un element la finalul vectorului
assign() Atribuie alt tip de date elementelor stocate în vector
